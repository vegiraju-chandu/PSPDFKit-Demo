Overview

requestRenderedImageForDocument:page:size:clippedToRect:annotations:options:priority:queueAsNext:delegate:

Requests a (freshly) rendered image from a specified document. Does not use the file cache. For options, see PSPDFPageRender. IF queueAsNext is set, the request will be processed ASAP, skipping the current queue.
